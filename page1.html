IMPERATIVE

Императивная "карусель". Может быть статически инициализирована на странице: в любой контейнер требуется добавить список ul с классом "_carousel" и внутрь его поместить элементы li с классами "_carousel__item". Внутри элемента списка помещается картинка c тегом img. Стиль у картинки может быть любой или не быть вообще. Также необходимо подключить файлы со стилями carousel.css и кодом carousel.js. После загрузки страницы такие карусели будут доступны через массив объектов window.carousels. В этом массиве каждый элемент представляет собой отдельную карусель на веб-странице в порядке их появления в потоке документа. Добавление новых элементов возможно через метод add объекта карусель. Количество элементов и текущий элемент в карусели доступны через свойства count и index соответственно. Через поле ввода и кнопку add внизу можно добавить новые изображения по URL. На следующей странице будет динамическое создание карусели.
